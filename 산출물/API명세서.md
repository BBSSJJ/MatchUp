# 유저(user)

---

- `**[POST]` 회원가입**
    
    
    | Description | 입력받은 소환사명을 서버로부터 받은 snsId, snsType과 함께 전송한다. |
    | --- | --- |
    | URL | api/users/signup |
    | Auth Required | No |
    
    | Paramater | Type  | Required | Description | Default |
    | --- | --- | --- | --- | --- |
    | body params  |  |  |  |  |
    | summonerName | String | true | 소환사명 |  |

        ```jsx
        {
        "id": 1,
        "riotTokenValue": "asdasdasdasd",
        "name": "갓상엽 #갓갓",
        "iconUrl": "thisisIconUrl",
        "level": "341",
        "tier": "",
        "leagueRank": "",
        "leaguePoint": "",
        ```

